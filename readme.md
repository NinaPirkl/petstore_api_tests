Petstore API Tests

Overview

This repository contains automated tests for the Petstore API using Java. The tests are written using JUnit 5 and RestAssured to validate various endpoints and ensure the correct functionality of the API.

Dependencies

JUnit 5: Used for writing and running unit tests.
RestAssured: A powerful library for testing REST APIs.
Jackson: For JSON processing and mapping.
Hamcrest: Used for more expressive assertions.

Setup

Clone the repository:
git clone https://github.com/yourusername/petstore-api-tests.git
Navigate to the project directory:
cd petstore-api-tests
Install dependencies using Maven:
mvn install
Run the tests: To run all the tests, execute:
mvn test
